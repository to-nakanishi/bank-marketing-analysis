銀行の顧客ターゲティング(SIGNATE提供：https://user.competition.signate.jp/ja/)  
　顧客の属性情報などから定期預金キャンペーンの反応率を予測  
1.結果  
　精度: 79%  
　モデル：AUC  
  
2.特徴  
　(1)体系的な特徴量エンジニアリング  
　(2)全18カラムの処理方針を分析的に設計  
　　 カテゴリ処理方法理由  
　　　・リスト符号化順序情報を保持(job, education, contact, poutcome)  
  　　・符号化バイナリ変数(marital　0/1)  
　　　・グループ化分布に基づくビニング(age, balance, day)  
  　　・グループ化(pdays：'[-1, 0,1~200, 201-400, 401~]')
    
  (3)Duration（通話時間）削除  
  　　初期モデルにおいてdurationが寄与度0.40で最高値だったが、因果が逆であるため削除。  
  　　（補足：成約した場合は内容について詳しく説明を行い、通話時間が長くなる。  
  　　つまり duration は「成約の結果」で「原因ではない」ため、通話時間を伸ばしても成約率は上がらない。）  
　　よって、削除により実装可能で効果的なモデルに変更  
  
3.技術  
  Python, pandas, scikit-learn, XGBoost, LightGBM  
  
4.ファイル  
　・データセット: SIGNATE参照  
　・bank_marketing.ipynb: 分析ノートブック  
　・submission.csv: 提出結果  
   
5.学び

　・クラス不均衡データ（陽性率11.7%）への対応  
　・「精度」と「ビジネス実装可能性」のバランス  
　・データドリブンな意思決定の重要性
